    Learn to build web3 apps

    Holochain is an open-source protocol for developing peer-to-peer applications.

    The protocol allows for highly modular development of local-first functionality that can be combined into fully featured cryptographically secure applications.

    Holochain is a key part of a new robust, resilient and sovereign technological infrastructure. You can learn more here.

    Outcomes

    At the end of the program, you can:

  • Make and distribute simple Holochain apps
  • Work as a Holochain developer
  • Understand the fundaments of Holochain app architecture

    Course Pre-requiste

    Basic Rust

    In order for you to receive full value from this course, you must already have experience coding in Rust.

    While most of a holochain app can be coded in Javascript using your preferred front end frameworks, Rust is required for the core Holochain functionality and integrity.

    Lectures
